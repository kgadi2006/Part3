# Part3 Amber & Oak Cafe

A fully responsive and interactive website for a coffee shop and bakery. Users can browse products, explore the menu, place orders, and simulate payments, all with smooth animations and a mobile-friendly design.

Features

Home Page: Hero section, welcome text, Explore Our Menu button and a search bar to filter menu items.

Menu Section: Grid layout with images, names and prices, hidden by default and revealed on click; scroll animations.

Products & Services Gallery: Interactive lightbox for viewing larger images with captions.

Order Now Page: Browse products, add items to a cart  and view totals.

Pay Now Page: Displays cart, simulates payment, clears cart on completion.

Responsive Design: Works on mobile and desktop, hamburger menu toggle on smaller screens.

Interactive Features: Smooth scrolling, scroll animations, live footer date/time and dynamic menu search.

HTML5, CSS, JavaScript

Usage

Open index.html in a browser.

Click Explore Our Menu to view the menu.

Use the search bar to filter menu items.

Add items to your cart on Order Now.

Visit Pay Now to view your cart and simulate payment.
